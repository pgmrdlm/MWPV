Thanks! Since your CategoryGrid is inside a Panel UserControl, and Panel is used in MainWindow, the flow looks like this:


MainWindow
└── Panel (UserControl)
    └── CategoryGrid (UserControl)
        └── DataGrid
So to refresh the category grid from MainWindow, you need to:

✅ Final Plan:
🔹 1. CategoryGrid.xaml.cs — Add a public Refresh() method

Edit
public void Refresh()
{
    var categories = CategoryService.LoadCatagories();
    CategoryDataGrid.ItemsSource = categories; // Name your DataGrid "CategoryDataGrid"
}
Make sure the DataGrid in CategoryGrid.xaml has:
